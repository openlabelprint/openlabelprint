Introduction

This page describes the requirements and identifies the many options for running Barcode4J.

Requirements

  • Java Development Kit (JDK) 1.3 or higher (available from http://java.sun.com/)
  • Download Barcode4J as described on:
    • the Download page, or
    • from CVS (requires Apache Ant 1.5 or higher, then follow directions on the Building page)

Running

Depending on how you will run Barcode4J, you will have differing dependencies. Click the links below for directions on using Barcode4J:
