Barcode4J Demo Applet

  • Instructions

Instructions

This little java applet demonstrates Barcode4J's output. Play around with it and see how it works. Try the different barcode formats by selecting a new one from the barcode format popup. If a combination of barcode type and message generates an error, move your mouse over the region where the barcode should have been. A little popup will show you the error message.

Java needed to display this Applet.

The source code for this applet can be found under "examples/demo-applet". The applet also serves as a demonstration of the new JavaBean-style API and of using Barcode4J in a AWT/Java2D environment.
